# 📦 Supply Chain Performance Analysis & Late Delivery Prediction

## Project Overview

The **Supply Chain Performance Analysis & Late Delivery Prediction** project is an end-to-end data analytics and machine learning solution developed using Python. It focuses on analyzing supply chain operations, identifying delivery delays, evaluating profitability, and predicting the risk of late deliveries using historical order data.

The project combines data preprocessing, exploratory data analysis (EDA), feature engineering, business intelligence, and predictive modeling to provide actionable insights for improving logistics and operational efficiency.

---

# Purpose

This project enables businesses to analyze supply chain performance, monitor delivery efficiency, identify operational bottlenecks, and predict orders that are likely to be delivered late. It provides a data-driven approach to improving customer satisfaction and optimizing supply chain operations.

---

# Tech Stack

The project was developed using the following tools and technologies:

- 🐍 **Python** – Data analysis and machine learning
- 📊 **Pandas** – Data cleaning and manipulation
- 🔢 **NumPy** – Numerical computations
- 📈 **Matplotlib** – Data visualization
- 📉 **Seaborn** – Statistical data visualization
- 🤖 **Scikit-learn** – Machine learning model development
- ⚖️ **SMOTE (Imbalanced-learn)** – Handling class imbalance
- 📓 **Jupyter Notebook** – Development environment

# Features & Highlights

## Business Problem

Supply chain organizations often face operational challenges due to delayed deliveries, inefficient shipping methods, and inconsistent profitability across different regions and customer segments.

Key questions include:

- Which regions experience the highest delivery delays?
- Which shipping modes perform the best?
- Which departments generate the highest profits?
- Which customer segments are most affected by delays?
- Can late deliveries be predicted before shipment?

---

# Goal of the Project

- Analyze supply chain performance through data visualization.
- Identify factors contributing to delivery delays.
- Measure profitability across business dimensions.
- Study customer and regional order patterns.
- Predict late delivery risk using Machine Learning.
- Generate actionable business insights for decision-making.

- Delivery Delay

Performed analysis on:
